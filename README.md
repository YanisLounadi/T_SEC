# T-SEC-902 — DFIR / Forensic Investigations

Projet d'investigation forensique Epitech. Le dépôt regroupera plusieurs scénarios
d'analyse d'artefacts forensiques (PCAP, dumps mémoire, images disque, malware),
chacun rangé dans son propre dossier sous `scenarios/`.

## Organisation prévue

```
T-SEC-902/
├── README.md          # ce fichier (index)
├── project.pdf        # sujet du projet
└── scenarios/         # un dossier par scénario (à venir)
```

## Conventions

- **Un dossier par scénario** sous `scenarios/NN-nom/`.
- **Artefacts** volumineux **non versionnés** (voir `.gitignore`).
- Analyse détaillée et rapport DFIR au sein de chaque scénario.

> Sujet complet : voir `project.pdf`.
